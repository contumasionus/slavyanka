# Славянка — Информационный сайт для деревенского магазина

Сайт-визитка для продуктового магазина «Славянка» (деревня Курилово). Проект создан для ознакомления жителей с информацией о магазине и его ассортименте.

## Архитектура

```
├── apps/
│   ├── backend/      — Backend API (Fastify + Prisma + PostgreSQL)
│   └── frontend/     — Frontend (Vue 3 + Vite)
├── packages/
│   └── shared-types/ — Общие типы TypeScript
├── turbo.json        — Конфигурация монорепозитория
```

## Технологии

| Стек                        | Описание                 |
| --------------------------- | ------------------------ |
| **Turborepo**               | Менеджер монорепозитория |
| **Fastify**                 | Backend API сервер       |
| **Prisma ORM**              | Работа с базой данных    |
| **PostgreSQL**              | База данных              |
| **Vue 3 + Composition API** | Frontend фреймворк       |
| **Pinia**                   | State management         |
| **Vue Router**              | Маршрутизация            |
| **Vite**                    | Сборщик                  |
| **TypeScript**              | Типизация                |

## Функциональность

### Публичная часть

- **Главная страница** — основная информация о магазине.
- **Каталог** — ознакомление с ассортиментом товаров.
- **О нас** — история и описание магазина.
- **Контакты** — информация о расположении и графике работы.

### Админ-панель

Отдельное приложение для управления контентом:

- **Управление товарами** — создание, редактирование и удаление товаров в ассортименте.
- **Управление категориями** — настройка структуры каталога.
- **Просмотр заказов** — просмотр поступивших запросов/заказов.
- **Управление пользователями** — контроль доступа к системе.

## Установка и запуск

### Предварительные требования

- Node.js >= 18.0.0
- PostgreSQL

### Установка

```bash
# Клонировать репозиторий
git clone <repository-url>
cd slavyanka-store

# Установить зависимости
npm install
```

### Настройка базы данных

Проект использует Prisma ORM. Настройте подключение к базе данных в `.env` файле внутри `apps/backend/`.

### Запуск

```bash
# Запуск всех приложений
npm run dev
```
